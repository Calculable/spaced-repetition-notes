# Materialien

Materialien führen zu einem Transparenzeffekt:

```swift
Text("Visit Singapore")
	.padding()
	.background(.thinMaterial)
```
![][image-1]

Folgende Materialien sind verfügbar

- `.ultraThinMaterial`
- `.thinMaterial`
- `.regularMaterial`
- `.thickMaterial`
- `.ultraThickMaterial`
