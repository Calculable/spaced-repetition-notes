# Element soll die ganze Verfügbare Breite verwenden
↔️

```swift
.frame(maxWidth: .infinity)
```


Häufig kombiniert man das auch mit einem Alighment:

```swift
Text("Hello, world")
    .frame(maxWidth: .infinity, alignment: .trailing)
```
