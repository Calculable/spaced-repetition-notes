# Modaler Dialog ohne Segue
🗒️

## Modaler Dialog anzeigen

```swift
present(selectionVC, animated: true, completion: nil)
```

## Modaler Dialog schliessen
So kann der Modale Dialog wieder geschlossen werden:

```swift

```
