# XcodeGen

## Konfigurationsfile
- Xcode Gen generiet ein SwiftProjekt anhand eines Konfigurationsfiles
- Das Konfigurationsfile ist ein YAML oder JSON
- Das File enthält Targets, Konfigurationen, Schemes, Build Einstellungen etc.

##  Vorteil
- `.xcodeproj` wird lokal generiert, muss also nicht mehr eingecheckt werden
- =\> Weniger Merge Konflikte
- `.xcodeproj` ist nicht wirklich Menschenlesbar
