# One-Sided Ranges
↔️

Nicht verwechseln mit Infinite Lists für Haskel

Erste drei Items eines Arrays auslesen:

```swift
let bigParts = characters[..<3]
```

Letzte drei Items eines Arrays auslesen:

```swift
let smallParts = characters[3...]
```
