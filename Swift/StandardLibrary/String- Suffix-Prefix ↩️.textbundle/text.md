# String: Suffix/Prefix ↩️

## Has Prefix

```swift
let result = name.hasPrefix("Prefix")
```

## Has Suffix

```swift
let result = name.hasSuffix("Suffix")
```

## Prefix

```swift
let result = name.prefix(3)
```


## Suffix

```swift
let result = name.suffix(3)
```

## Zusammenfassung
- HasPrefix / HasSuffix
- Prefix / Suffix
