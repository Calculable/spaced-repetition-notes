# localizedStandardCompare
🎱

- Es ist ein Funktion auf dem String, welcher wie der Finder sortiert
- Ist der Standart Comperator für String-Vergleiche
- Beispiel:

	User 1
	User 2
	User 11
	User 12
	User 110

- Alphabetisch wäre die Reihenfolge anderst!
